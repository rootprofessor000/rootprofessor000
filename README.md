</a>

> **🔍 ReconX** — A modular **async reconnaissance & vulnerability-scanning framework** for authorized penetration testing and bug bounty. Chains **33 modules** (subdomain/DNS/WHOIS recon, port & TLS scanning, WAF/CMS fingerprinting, and detection of SQLi, XSS, SSRF, CORS, LFI, command injection, exposed secrets & cloud buckets) through one `asyncio` pipeline — with HTML/JSON/Markdown reports, Slack/Discord/webhook alerts, a SQLite scan-history store, **177 tests**, CI, and Docker.
